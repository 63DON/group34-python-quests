# Group 25 - Python Quests

Our group's solutions for the **Learn Python by Coding** quests. The goal is to build a strong foundation in programming, covering variables, input, math, conditionals, loops, and functions across 6 levels.

## Repository structure

```
group25-python-quests/
+-- quests/
|   +-- quest_05_echoing_cave.py
|   +-- quest_10_architects_blueprint.py
|   +-- quest_15_nested_riddle.py
|   +-- quest_20_even_number_forager.py
|   +-- quest_29_code_breaker.py
|   +-- quest_30_reflective_scribe.py
+-- README.md
```

Each member contributes one quest per level so the group covers at least 4 tasks in every one of the 6 levels.

## How to run

Make sure Python is installed:

```bash
python --version    # or: python3 --version
```

Run any quest:

```bash
python quests/quest_05_echoing_cave.py
```

Some quests (Q10, Q15, Q29) ask for input - just type your answer when prompted.
